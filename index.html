bunnydns

A Python SDK for the Bunny.net DNS API.

Installation

pip install bunnydns

Quick Start

from bunnydns import BunnyDNS

client = BunnyDNS(access_key="your-api-key")
zones = client.list_dns_zones()
for zone in zones.items:
    print(zone.domain)
