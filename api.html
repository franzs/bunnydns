Client

class bunnydns.BunnyDNS(access_key, base_url='https://api.bunny.net', timeout=30)[source]

Client for the Bunny.net DNS API.

Parameters:

  • access_key (str) – Your Bunny.net API access key.

  • base_url (str) – Override the default API base URL (useful for testing).

  • timeout (int) – HTTP request timeout in seconds.

Example

>>> client = BunnyDNS(access_key="your-api-key")
>>> zones = client.list_dns_zones()
>>> for zone in zones.items:
...     print(zone.domain)

Update the settings of an existing DNS zone.

Only non-None parameters will be included in the update request.

Parameters:

  • zone_id (int) – The ID of the DNS zone to update.

  • custom_nameservers_enabled (bool | None) – Enable or disable custom nameservers.

  • nameserver1 (str | None) – The first custom nameserver.

  • nameserver2 (str | None) – The second custom nameserver.

  • soa_email (str | None) – The SOA email address for the zone.

  • logging_enabled (bool | None) – Enable or disable query logging.

  • logging_ip_anonymization_enabled (bool | None) – Enable or disable IP anonymization in logs.

  • log_anonymization_type (LogAnonymizationType | None) – The type of log anonymization to apply.

  • certificate_key_type (CertificateKeyType | None) – The private key type for wildcard certificates.
